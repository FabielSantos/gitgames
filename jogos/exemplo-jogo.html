Descrição

Exemplo Jogo é um jogo de ação fictício usado como modelo para a estrutura das páginas do Git Games. Aqui você encontra a mesma organização que terão todas as páginas de jogos: descrição, dicas, links e categorias.

Dicas e truques

  • Use as teclas de atalho para acessar o inventário mais rápido.
  • Guarde recursos no início para desbloquear a arma secreta no nível 5.
  • Explorar os cantos do mapa revela easter eggs.
